# Iris Flower Classification 

##  Overview
This project builds a machine learning model to classify iris flower species (Setosa, Versicolor, Virginica) based on their features.

##  Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

##  Task Details
- Loaded and explored dataset
- Trained classification model
- Evaluated using accuracy and confusion matrix

##  Result
Achieved good accuracy on test data.
